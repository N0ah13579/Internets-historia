Internets histroia

Arpanet

Arpannet (Avanced Research Projects Agency Network) utveklades av det amerikanska DARPA under kalla kriget och vart klar år 1969. Vid år 1971 blev det första e-postmedelandet skickat.

FTP

FTP (File Transfer Protocol) är ett av det tidigaste populära filöverföringsprotkollen för internet.

Telnet

Telnet är till för textbaserad inloggning och kommunikation.

SMPT

Simple Mail Transfer Protocol (SMPT) är det vanligaste kommunikationprotokollet för e-post.

HTML

HTML (HyperText Markup Language) är ett märkespåk för hypertext och är den grundläggande starndaren för WWW(World Wide Web) webbsidor skrivs i HTML och sen överförst till internetet med HTTP.

HTTP

HTTP (HyperText Transfer Protocol) används som det kommunikationprotokollet för att överföra HTML-sidor webservrar till webbkileter.

Webläsare

Är ett program man laddar ner på sin enhet eller är redan på en enhet när man köper den och webläsare är ett program för att hämta, tolka och återge dokument programmerade i HTML och XHTML som har laddats upp webservrar på internet.

CSS

CSS (Cascading Style Sheets) är det språket som beskriver för en webläsare hur presationsstillen är strukterar i ett dukoment, ett exemple är typsnitt och färg.

XML

XML (Extensible Markup Language) är ett märkspåk som är universellt och en förenklad efterträdare till SGML.
